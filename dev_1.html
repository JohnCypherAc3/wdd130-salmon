Description of the Run

The Terraria Calamity mod true melee sub-class is one of the more difficult sub-classes in the mod to play. The basis of the class is using only weapons that do true melee damage (a damage type added in the Calamity mod) which is done by sword type weapons that do not launch projectiles.

The difficulty of using this class is that many enemies and bosses deal contact damage while to deal damage, the player must be close enough to the enemies to hit it with a small sword. Because of this, the sub-class’s damage is not consistent with many other classes or sub-classes though when able to hit the bosses, the damage is usually higher than the other sub-classes.

The run starts with the player having some of the smallest weapons known to man being their best options. Many of these early game weapons are extremely difficult and dangerous to use and do not have good compatibility with the bosses being fought as they move around quite often.

Later through the playthrough the player is able to get much larger weapons and even weapons that launch projectiles when they hit an enemy (while they do launch projectiles, the requirement to hit with the sword itself still applies due to the projectiles launching when the sword makes contact.

The end game introduces enchantments through the defeat of the boss Supreme Calamitas which allows for the “Tainted” enchant which changes any weapon into a true melee weapon the drawbacks being that the range is usually much less than the original weapon, though if the challenge is only true melee damage, this is perfect for end game gear.

Overall, this challenge is difficult but fun if done on the right mode (Don't try this challenge if you are not used to the bosses). Though there are some bosses that seem almost impossible, the whole game is able to be completed using the sub-class of true melee. Early in the game it may seem impossible but in the latter stages it is quite enjoyable and as such, if you ever feel like having a challenge in Terraria Calamity mod, this is quite an enjoyable option.
